Demo App

This is a demo application to show the authentication and authorization using OAuthServer built in spring-boot.

Click on the Login button to sign-in using the authorization_code flow in the OAuth2 spec.

Welcome {{user.firstName + ' ' + user.lastName}}

You can now navigate to employees page.

Employees page will display a list of all the employees in the database. To retrieve this data a valid access token / jwt is required by the backend.

The Add button on the employees page will be visible only to users with ADMIN_ROLE.
